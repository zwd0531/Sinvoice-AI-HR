# 智慧招聘 AI HR 助理系统 Demo

基于思必驰AI办公本的企业招聘场景智能系统演示版本

## 启动

```bash
npm install
npm run dev
```

## 页面

| 路由 | 功能 |
|------|------|
| `/` | 首页 — 产品介绍与功能导航 |
| `/hr` | HR助手 — 智能面试辅助与报告生成 |
| `/candidate` | 候选人复盘 — 视频回放、字幕、竞争力分析 |
| `/resume` | 简历筛选 — 看板式智能简历评分与流转 |
| `/ai-interview` | AI面试官 — 模拟面试候选人视角与HR大盘 |
| `/chatbot` | 政策咨询 — HR政策问答聊天机器人 |

## 技术栈

- Next.js 16 App Router + React 19
- Tailwind CSS v4
- framer-motion v12（动画）
- recharts v3（数据可视化）
- react-markdown v10（Markdown 渲染）
- shadcn/ui 组件库
